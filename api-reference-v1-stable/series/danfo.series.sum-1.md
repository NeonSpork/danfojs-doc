# danfo.Series.sum

Return the sum of the values for the requested axis.

This is equivalent to the method numpy.sum.

            **parameter:** 

            **return:** Series

**Example**

```javascript
let data1 = [30, 40, 3, 5]
let sf = new Series(data1)
sf.sum()
```
