# DataFrame.copy

danfo.DataFrame.**copy**()

## **Examples**

{% tabs %}
{% tab title="Node" %}
```javascript
const dfd = require("danfojs-node")
let data = {"A": [-20.1, 30, 47.3, -20],
            "B": [34, -4, 5, 6], 
             "C": [20, -20, 30, -40]}
let df = new dfd.DataFrame(data)
let new_df = df.copy()
new_df.print()
```
